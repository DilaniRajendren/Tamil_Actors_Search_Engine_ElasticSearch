# Tamil_Actors_Search_Engine_ElasticSearch

The repo contain more than 100 actors scrap from "https://ta.wikipedia.org/wiki" and stored at Actors_Scrapped.csv(Actors_Scrapped.json) file. Some of the metadata such as  "நடிகர்கள்","அறிமுகஆண்டு" "குறிப்பிடத்தக்க படங்கள்"are in English. For scrapping the data from websites scraping_data.py is used.
